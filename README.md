# Location Tracker

## Running Backend Server 
   ```cd api```

   ```npm install```
   
   ```node server.js```

## Running Client Server
  ```cd client``` 

  ```npm install```
  
  ```npm start```

## Description
   Tracking Start as soon as you open website
   
   There's a dropdown in rightmost corner from where you can change time interval after which data gets mapped.
